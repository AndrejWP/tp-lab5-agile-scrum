# Система управления задачами для студентов

Это веб-приложение, разработанное специально для студентов университетов, чтобы помочь им организовать учебный процесс. Приложение позволяет создавать списки задач, привязывать их к конкретным предметам и устанавливать дедлайны. Пользователи могут отслеживать прогресс выполнения курсовых, лабораторных и домашних заданий в реальном времени. В системе реализована функция уведомлений, которая напоминает о приближающихся сроках сдачи работ. Также предусмотрен календарь, где визуально отображается загруженность студента по дням и неделям. Для повышения мотивации внедрена система геймификации с достижениями за вовремя закрытые сессии. Проект направлен на снижение стресса во время учебы и улучшение тайм-менеджмента.

## Sprint Review
**Итоги спринта 1:**
В ходе спринта команда взяла в работу 5 задач.
- **Полностью завершены:** "Дизайн главной страницы" и "Регистрация пользователей".
- **В процессе:** "Создание новой задачи" (возникли сложности с API).
- **Не успели начать:** "Календарь дедлайнов" и "Уведомления в Telegram".

Цель спринта по запуску базовой версии выполнена частично (есть вход, но нет функционала задач).

## Sprint Retrospective
**Что прошло хорошо:**
- Дизайнер быстро подготовил макеты.
- Архитектура базы данных для пользователей спроектирована верно.

**Что пошло не так:**
- Недооценили сложность связки фронтенда и бэкенда при создании задач.
- Потратили время на настройку сервера, которое не учли в планировании.

**План улучшений:**
- В следующем спринте закладывать время на DevOps-задачи.
- Разбивать крупные задачи (как "Создание задачи") на подзадачи поменьше.
